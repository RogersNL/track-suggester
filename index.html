, you sound like you want to sell your soul to Microsoft. Check out C#!:

C#

C# is a widely used language among large buisnesses. It has been around for a while and was originally designed to be easy to learn and use. It's a high-level language so it doesn't require a lot of complex tasks but it's still not as high-level as others. It has the backing of Microsoft, so if you're looking to work there this would be a great place to start!

Students at Epicodus will cover the following:

, you seem interested in a diverse set of skills that you can use in apps and other applications. Check out JAVA!:

JAVA

JAVA is by far one of the most popular and versatile of the programming languages. Its use ranges from Android smartphone apps to complete computer applications. Due to its versatility, being able to run the same code on two devices that are nothing alike, it is commonly used in enterprise-level firms.

Students at Epicodus will cover the following:

  • Behavior-Driven Development
  • Web Applications
  • Database Basics
  • Advanced Topics in JAVA

JAVA is commonly used in everything from web applications to backend services to mobile applications to embedded systems.

, you're probably looking to work on the backend, enableing websites to have a solid foundation and functionality. Check out PHP!:

PHP

PHP is best known for its overwhelming use in backend structure for websites. With 80% of websites taking advantage of its 'server-side' scripting capabilities, it is a great first language to learn. Some notable sites that utilize PHP are WordPress, Facebook, and Tumblr. PHP allows the user to collect form data, generate dynamic page content, send and receive cookies, and much more!

Students at Epicodus will cover the following:

  • Object-Oriented Design Basics
  • Test-Driven Development with PHPUnit
  • Database-Basics with SQL

PHP is a must for backend website building.

, the design and user experience is important to you. Check out RUBY!:

RUBY

RUBY is a fast-paced, and even FUN, language that utilizes its innovative rails-network to quickly create beautiful, functional websites. With RUBY, there's often multiple 'best' solutions to a problem that allows for creativity from the programmer. It is a high-level language that focuses more on the backend, allowing sites have account creation, information entry, and interactive dynamic content. Young companies and start ups often use RUBY for their websites.

Students at Epicodus will cover the following:

  • Behavior-Driven Development
  • Object-Oriented RUBY
  • Database Basics
  • Advanced Databases with Acive Record

RUBY has a steep learning curve but allows for diversity and speed in website building.
